# Mission Mod
- Destroy the nukes before the timer reaches 0
- Multiplayer co-op campaign mission

## Download
1. Start Warzone 2100. Click **Options**
2. Click "Open Configuration Directory"
3. Download [`📦4c-Cooperative_1.wz`](https://github.com/aco4/WZmodMission/raw/refs/heads/main/map/4c-Cooperative_1.wz). Put in `📁maps/`
4. Download [`📦WZmodMission-v0.0.1.zip`](https://github.com/aco4/WZmodMission/releases/latest). Put in `📁mods/4.7.0/autoload/`
5. Restart Warzone 2100

## Required Game Settings
- Team A: Mission Mod AI
- Team B: All other players

## Difficulty
- **Insane**: 30 minute timer
- **Hard**: 45 minute timer
- **Medium**: 1 hour timer
- **Easy**: No timer
